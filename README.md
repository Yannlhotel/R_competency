# Machine Learning avec R

Ce dépôt contient le contenu du cours de Machine Learning II utilisant R. 
Le cours couvre différents aspects de l'apprentissage automatique, notamment :


1. **Gestion des Données Manquantes**
   - `01_MissingData.R`

2. **Classification Bayésienne**
   - `02_1_Bayes_functions.R`
   - `02_2_LDA_QDA_Diabetes.R`
   - `02_Bayes_functions.R`

3. **Classification et Analyses Diverses**
   - `03_NB_Iris.R` - Classification Naive Bayes sur le dataset Iris
   - `03_QuitJob.R` - Analyse de départ d'employés
   - `03_Titanic.R` - Analyse du dataset Titanic

4. **Support Vector Machines**
   - `04_SVC_College.R` - SVM appliqué aux données College
   - `04_test_and_run.R` - Tests et exécution
   - `05_Non_linear_SVM.R` - Implementation of SVM with non linear kernel
   


Pour lancer ces scripts R , il faut :
- R et RStudio
